# Missions Power Ministries

Website for Missions Power Ministries, Inc. — the ministry of Pastor Terry
Wong, author of *Missions Power*. Built with Next.js and Tailwind CSS,
deployed on Vercel.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deployment

Pushes to `main` deploy automatically via the Vercel GitHub integration.
